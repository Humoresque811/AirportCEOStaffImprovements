[h1]Description:[/h1]
AirportCEO Staff Improvements is a code based mod that currently only adds a way to train all hired staff. However, in the future it could host more options. Config is accessed via the F1 key.

[h1]Details:[/h1]
Features:
- New button to train all (hired) staff that can be trained, and simple system around it
